## Privacy Policy related to bot

**What data do we store?**<br>
We store only server id and no other data

**How and Why do we need the data?**<br>
We need the data for prefix command so that users can set custom prefix in their server.

**Do we share the data?**<br>
We do not share any data with anybody, Only the creator of the bot has access to the data.

**How can users contact us if they have concerns about the bot?**<br>
They can dm Bravestone#5405 or join the support server for the same.

**How do we get the data removed?**<br>
You can use prefix reset command which removes the server id from our database.
