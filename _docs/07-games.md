### command description

| Command              | Aliases          | Uses             | Description
| ---------------------| -----------------|------------------|----------------------------------|
| **hangman**          | hg               | $hangman         | Starts a game of hangman         |
| **tictactoe**        | ttt              | $tictactoe <user>| Starts a game of tictactoe       |
| **trivia**           | -                | $trivia          | Starts a game of trivia          |
| **rps**              | -                | $rps             | Starts a game of rps             |
| **wordhunt**         | wh               | $wordhunt        | Starts a game of wordhunt        | 
| **scramble**         | sc               | $scramble        | Starts a game of scramble        |
| **passthebomb**      | ptb              | $passthebomb     | Starts a game of passthebomb     |  
| **guessthepokemon**  | gtp              | $guessthepokemon | Starts a game of guessthepokemon |
| **guessthemovie**    | gtm              | $guessthemovie   | Starts a game of guessthemovie   |
| **bowling**          | -                | $bowling         | Starts a game of bowling         |
